# Unturned Scripts
Datamining Unturned Scripts on every **preview** build.

## What are Scripts?
Unturned Scripts are [Unity Scripts](https://docs.unity3d.com/6000.5/Documentation/Manual/intro-to-scripting.html) that can be used to add functionality when modding for Unturned. Scripts in this project have been stripped of all non-essential dependancies to make it as easy as possible to include in your Unity project.

This [Steam Guide by Olie304](https://steamcommunity.com/sharedfiles/filedetails/?id=1086881979) explains how to use these Scripts.
